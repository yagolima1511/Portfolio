Projeto AlphaTec

Juntamente com meus colegas de faculdade, desenvolvemos o Projeto AlphaTec. O projeto do carrinho autônomo com Arduino é uma aplicação simples de robótica, onde o objetivo principal é criar um veículo que detecte obstáculos e mude de direção automaticamente usando um sensor ultrassônico. Aqui está uma explicação didática e direta sobre os principais componentes e funcionamento:

  • Arduino: É o "cérebro" do carrinho, responsável por processar as informações do sensor ultrassônico e controlar os motores.
  • Sensor Ultrassônico (HC-SR04): Detecta obstáculos à frente, medindo a distância usando ondas sonoras.
  • Motores e Ponte H (L298N): Movem o carrinho e permitem controlar o sentido de rotação (para frente, para trás ou virar).
  • Baterias/Pilhas: Alimentam o Arduino e os motores.
  • Estrutura do Carrinho: Uma base com rodas onde os componentes são fixados
